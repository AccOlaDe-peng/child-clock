# 儿童时钟应用

一个专为儿童设计的简单时钟应用，具有友好的界面和实时时间显示。

## 🎯 功能特点

### 🕐 主时钟界面

- **大字体数字时钟** - 清晰显示当前时间，支持上午/下午标识
- **实时更新** - 每秒自动更新时间显示
- **日期信息** - 显示月/日、星期和闹钟设置信息
- **状态指示** - 电池图标显示系统状态

### ⏰ 倒计时定时器

- **圆形进度条** - 直观显示倒计时进度
- **多种时间预设** - 5 分钟、10 分钟、15 分钟、30 分钟、1 小时、2 小时
- **完整控制** - 开始、暂停/继续、停止功能
- **时间到提醒** - 倒计时结束时弹出提醒对话框
- **实时显示** - 分:秒格式显示剩余时间

### ⚙️ 设置界面

- **闹钟设置** - 使用时间选择器设置闹钟时间
- **定时器设置** - 快速设置倒计时时间
- **时间格式** - 12 小时制/24 小时制切换
- **语言设置** - 中文/English 语言切换
- **主题设置** - 浅色/深色主题切换

### 🎨 界面特色

- **儿童友好设计** - 浅蓝色主题，清晰易读
- **等宽字体** - 模拟数字时钟显示效果
- **底部导航** - 三个界面间轻松切换
- **响应式布局** - 适配不同屏幕尺寸
- **直观操作** - 大按钮设计，易于操作

### 📱 多平台支持

- **Web 浏览器** - Chrome、Safari、Firefox 等
- **移动设备** - Android、iOS（需要相应开发环境）
- **桌面应用** - macOS、Windows、Linux

## 🚀 快速开始

### 在 Web 浏览器中运行（推荐）

```bash
flutter run -d chrome --web-port=8080
```

然后在浏览器中打开：http://localhost:8080

## 📖 使用说明

### 🕐 主时钟界面

- 查看当前时间和日期
- 显示闹钟设置信息
- 实时更新时间显示

### ⏰ 倒计时定时器

1. **设置时间**：点击"设置时间"按钮选择预设时间
2. **开始倒计时**：点击"开始"按钮启动倒计时
3. **暂停/继续**：点击"暂停"按钮暂停，再次点击继续
4. **停止倒计时**：点击"停止"按钮结束倒计时
5. **时间到提醒**：倒计时结束时自动弹出提醒

### ⚙️ 设置界面

1. **闹钟设置**：点击设置闹钟时间，支持时间选择器
2. **定时器设置**：快速设置倒计时时间
3. **时间格式**：切换 12 小时制/24 小时制显示
4. **语言设置**：选择中文或 English 界面语言
5. **主题设置**：切换浅色/深色主题

### 🎮 操作提示

- 使用底部导航栏切换不同界面
- 所有按钮都有清晰的图标和文字说明
- 设置更改会立即生效
- 支持热重载，代码修改后自动更新界面

## 🛠️ 开发环境问题解决

### 如果遇到 Xcode 错误

如果您看到 `xcrun: error: unable to find utility "xcodebuild"` 错误，这是因为没有安装 Xcode。解决方案：

1. **使用 Web 版本**（推荐）：

   ```bash
   flutter run -d chrome
   ```

2. **安装 Xcode**（如果需要 macOS 版本）：
   - 从 App Store 安装 Xcode
   - 运行：`sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer`
   - 运行：`sudo xcodebuild -runFirstLaunch`

### 如果遇到 Android 错误

如果您看到 Android 相关错误，解决方案：

1. **使用 Web 版本**（推荐）
2. **安装 Android Studio**：
   - 下载并安装 Android Studio
   - 配置 Android SDK
   - 运行：`flutter doctor` 检查配置

## 🆘 常见问题

**Q: 为什么选择 Chrome 而不是 macOS 版本？**
A: 因为您的系统没有安装 Xcode，Chrome 版本不需要额外的开发工具链，可以直接运行。

**Q: 如何停止应用？**
A: 在终端中按 `Ctrl+C` 停止应用。

**Q: 如何重新启动应用？**
A: 运行 `flutter run -d chrome` 命令。

**Q: 倒计时定时器如何使用？**
A: 点击"设置时间"选择预设时间，然后点击"开始"启动倒计时。可以随时暂停、继续或停止。

**Q: 闹钟功能如何工作？**
A: 在设置界面点击"闹钟设置"，选择时间后，当到达设定时间时会弹出提醒对话框。

**Q: 如何修改界面语言？**
A: 在设置界面点击"语言设置"，选择中文或 English 即可。

**Q: 应用支持哪些时间格式？**
A: 支持 12 小时制（上午/下午）和 24 小时制，可在设置中切换。

**Q: 如何自定义倒计时时间？**
A: 目前支持预设的 6 种时间（5 分钟到 2 小时），点击"设置时间"按钮选择。

**Q: 应用数据会保存吗？**
A: 当前版本设置不会持久保存，重新启动应用后会恢复默认设置。
